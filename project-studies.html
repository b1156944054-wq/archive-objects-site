Project Studies

Scenarios that show how the objects enter real spaces.

Before full commercial case studies are available, project studies explain where each object belongs and why a client would commission it.

这不是假装已有大案例，而是用“项目研究”把产品和客户场景连接起来。

Study 001

Villa Entrance Bench

A sculptural bench placed in a private villa entrance. Its role is not just seating; it becomes the first memory point after the door opens.

  • Object: Bench
  • Client: private owner
  • Material route: metal + mineral surface
Concept ready · suitable for first commission
Study 002

Boutique Hotel Lobby Object

A low table and light object system for a boutique hotel lobby, creating an art-like waiting zone without losing function.

  • Object: Table + Light
  • Client: hotel / operator
  • Material route: stone-like surface + embedded light
Hospitality scenario · scalable
Study 003

Gallery Cabinet Object

A cabinet designed for display, concealment, and ritual. It can hold jewelry, art books, fragrance, ceramic works, or private collection pieces.

  • Object: Cabinet
  • Client: gallery / collector
  • Material route: glass + metal + dark interior
Collector scenario · premium route
